# Barle Kitchen & Pub

Welcome to the Barle Kitchen & Pub project! This repository contains the source code for a modern and elegant web application designed to showcase the unique dining and entertainment experience offered by Barle Kitchen & Pub.

## Features

- **Responsive Design**: Optimized for all devices, ensuring a seamless user experience.
- **Dynamic Pages**: Includes sections for About, Contact, Events, Menu, and Reservations.
- **Reusable Components**: Built with modular and reusable components for scalability.
- **Modern UI**: Styled with the latest design trends to provide a visually appealing interface.

## Project Structure

The project is organized as follows:

```
app/
  globals.css       # Global styles
  layout.tsx        # Root layout component
  page.tsx          # Home page
  about/            # About page
  contact/          # Contact page
  events/           # Events page
  menu/             # Menu page
  reservation/      # Reservation page
components/
  ui/               # UI components (e.g., buttons, cards, forms)
  navigation.tsx    # Navigation bar
  footer.tsx        # Footer component
hooks/              # Custom React hooks
lib/                # Utility functions
public/             # Static assets
styles/             # Additional styles
```

## Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/barle-kitchen-pub.git
   ```

2. Navigate to the project directory:
   ```bash
   cd barle-kitchen-pub
   ```

3. Install dependencies using pnpm:
   ```bash
   pnpm install
   ```

4. Start the development server:
   ```bash
   pnpm dev
   ```

5. Open your browser and visit `http://localhost:3000`.
